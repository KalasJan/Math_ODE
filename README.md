# Analytical ODE Solvers (SymPy Framework)

A structured framework of Python templates designed to analytically solve, numerically evaluate, and visualize Ordinary Differential Equations (ODEs) using open-source tools instead of rigid academic software.
 
## Key Features
- **Classification Coverage:** Templates for both 1st and 2nd-order Ordinary Differential Equations (ODEs).
- **Homogeneity Scenarios:** Solutions for both homogeneous and non-homogeneous equations.
- **Boundary & Initial Values:** Frameworks handling equations without conditions, with Initial Value Problems, and Boundary Value Problems.
- **Hybrid Pipeline:** Connecting symbolic math solving (`SymPy`) with fast numerical array evaluation (`NumPy` via `lambdify`) and automated plotting (`Matplotlib`).

## Tech Stack
- **SymPy** – For symbolic mathematics and analytical equation solving.
- **NumPy** – For vectorization and numerical evaluation of the analytical results.
- **Matplotlib** – For professional plotting with LaTeX mathematical formatting.
